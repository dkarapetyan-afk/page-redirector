Page Redirector Rules

Manage your custom URL redirects here.

Add / Edit Rule

Only evaluate the bytecode script if the requested URL matches this regular expression.

Regex Syntax: Uses standard JavaScript regular expressions.

Examples:

  • ^https://example\.com/(.*)$
  • example\.org/page\.php\?id=(\d+)

Wildcard Syntax: Use an asterisk (*) to match any characters.

Example: https://example.com/images/*

Bytecode Syntax: Use the stack-based language. Emit a string and call redirect.

Example: host "example.com" eq [ "https://newdomain.com" redirect ] call-if
